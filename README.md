# Constrained Refactor Gauntlet

An RL environment where an agent refactors a legacy Python codebase while obeying cascading engineering rules. Reward = CodeScore × ComplianceScore.

## Features

* OpenEnv compliant API
* Complex rule-based constraints
* Rewards based on code quality and compliance

## Setup

1. Clone the repository.
2. Install dependencies: `pip install -r requirements.txt`
3. Start the server: `python server.py` (or check specific configuration files)
